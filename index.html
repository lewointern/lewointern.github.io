乐我 - 云分析平台

Lewo – Cloud Analytics Platform

Lewo云分析平台是针对大数据平台和应用优化的一体化数据中心PaaS平台。Lewo支持公有云和私有云,可以方便的管理跨节点运行的集群容器系统 (Hadoop, HAWQ, Spark) 和各种应用; 提供数据分析,查询和存储服务, 并提供自动伸缩,自恢复,滚动升级和管理监控服务

Lava 云操作系统

Lava 云操作系统提供数据中心的节点资源管理和容器编排功能,支持各种灵活的容器管理和部署模式,可以方便的实现弹性伸缩、自恢复和滚动升级, 并提供可视化管理和监控功能。轻松得为您的应用提供 24*7 的高可用服务。

Lewo 云分析服务

Lewo 云分析服务基于 HAWQ-DB 和 Spark,提供结构化和非结构化数据分析服务。HAWQ-DB 是一个完全兼容 ANSI-SQL 标准的高性能原生 Hadoop MPP++分析型数据库。提供对 Hadoop 上 PB 级数据的高性能交互式查询。 并且提供对主要 BI 工具的描述性分析支持。Spark 提供一个统一的机器学习,数据挖掘的数据管理平台。

Lewo 查询服务

提供对万亿级对象存储及毫秒级快速键值查询响应能力。适用于内容管理、小文件以及图像存储查询。基于 HBase 和 HDFS。

Lewo 存储服务

Lewo 存储服务为简化有状态应用(比如 Postgres,MySQL 等)的管理和部署而设计,为有状态应用提供标准文件系统接口和高性能持久化存储。

Lewo 管理监控服务

Lewo 管理监控服务管理整个平台组件的用户、验证和基于角色的授权,和计费服务。并且提供日志管理与监控服务。




Lewo 核心特性

一体化数据与应用 PaaS 平台

Lewo 是世界上第一个集成了 PaaS 和数据平台的一体化数据中心操作系统。 Lewo 突破性的解决了有状态的数据平台和无状态应用无缝结合到一个数据中心操作系统的挑战,使得整个数据中心的运维、平台及应用开发变的无比简化。

完善的数据平台服务

Lewo 数据分析服务完全兼容 SQL-92, SQL-99, SQL-2003 以及 OLAP 扩展。 支持标准 JDBC 和 ODBC 连接,支持 Tab eau 等各大主流 BI 可视化工具和 ETL 工具。并提供完善的机器学习和数据挖掘功能。并同时支持各种非结构 化数据的处理。
Lewo 查询服务通过高效的分布式索引提供对亿万级对象的低延迟高吞吐的查询服务。Lewo 数据分析服务和查询服务的结合使得您轻松管理您的所有类型的数据。

高可扩展性

Lewo 云分析平台可以支持上万节点集群,使得您不再需要为数据的快速增长和用户数的快速增长而频繁的改变数据中心基础架构。在线增加和移除节点毫秒级可以完成 。

简单易用性

学习和使用 Lewo 非常简单,基本上所有的操作都已经被平台自动化。Lewo 提供可视化界面、命令行以及 API 接口供外部使用。
